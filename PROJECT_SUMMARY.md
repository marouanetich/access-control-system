# Access Control System Dashboard

## Overview
This project is a React-based security dashboard designed to simulate and manage access control systems. It features a realistic interface for monitoring security status, managing access points, simulating attacks, and viewing security logs.

## Features
- **Dashboard**: Real-time overview of security metrics and system status.
- **Access Control**: Interface for managing door locks and secure zone access.
- **Attack Simulation**: Tools to simulate various security threats (e.g., brute force, biometric spoofing) to test system resilience.
- **Security Logs**: Comprehensive log view of all system activities and alerts.
- **System Lockdown**: Global lockdown mode with visual and audio alerts in response to critical threats.

## Technology Stack
- **Frontend**: React (v19) with Hooks
- **Build Tool**: Vite
- **Styling**: Tailwind CSS (inferred from class names like `fixed inset-0`)
- **Icons**: Lucide React
- **Charts**: Recharts

## Setup and Installation

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
